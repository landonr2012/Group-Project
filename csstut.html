Basic CSS Tutorial

What is CSS?

Cascading Style Sheets (CSS) is a markup language used to specify how web page elements should be rendered in a browser. It is most commonly used to set the visual style of web pages and user interfaces written in HTML and XHTML, but we will only be concerned with HTML. CSS is a seperate language from HTML even thought they both complement each other. HTML specifies the content of each web page element, while CSS indicates how that content should be displayed. You create CSS code for an HTML document by writing style rules.



•Style rules - Lines or blocks of CSS code that specify the presentation of web page elements.



Basic Concepts CSS is Built On


Properties

A property is an aspect of the appearance of a web page element that can be modified or changed. For example, you might want to center the text of the <p> element by using the text-align property. CSS property names include hypens (-) instead of spaces.

Declarations

Declarations are the building block of all CSS code. They are a combination of a property name and a value. A declaration is enclosed within a pair of curly braces({ }). Inside the braces, property names are followed by a colon (:), the value and a semicolon (;) at the end.

Selectors

Selectors are used to associate declarations with web page elements. CSS allows the use of many different selectors.

Type Selectors

Type selectors are some of the most common selectors used and apply to all HTML elements of the same type.

Example of a style rule using a type selector

p {
text-align: center;
background-color: ivory;
}





ID Selectors

ID selectors are used when you make changes that only affect a single element. Do this by giving the element a unique identifier in its opening tag.

Assigning an ID to an element


<h1 id="name">Lakeland Reeds Bed & Breakfast</h1>


When applying a style rule to an ID selector in the CSS stylesheet, simply put a pound sign (#) in front of the ID.


Applying a style rule to an ID selector


#name {
text-align: right;
}





Class Selectors

Used when applying a style rule to multiple elements in a HTML document. A class selector can be used many times unlike the ID selector which is used once.

Assigning a class to an element


<h1 class="name">Lakeland Reeds Bed & Breakfast</h1>


When applying a style rule to the class selector in the CSS stylesheet, add a period (.) in front of the class name.


Applying a style rule to an class selector


.name {
text-align: right;
}



Types of Style Sheets



Embedded Style Sheet

In order to affect the presentation of elements in a single web page, you need to create and embedded style sheet. An embedded style sheet consists of CSS style rules entered in the head of an HTML document. When you add CSS styles to an HTML document, you must nest them inside the <style>.


Creating an Embedded Style Sheet

<head>

<style>

body {

background-color: red;
text-align: left;

}

<style>

<head>


External Style Sheets

Embedded style sheets are great for single web pages, but what if you wanted the same styles on multiple pages? You would then use an external style sheet, which is a seperate file containing the style rules that multiple web pages can link to. To incorporate an external style sheet, you need to use the link element to refernce the location of the CSS file.


Linking to an External Style Sheet


<link rel="stylesheet" href="styles.css">


In this example, the rel attribute indicates the type of content in the linked file. The href attribute specifies the filename of the style sheet.
